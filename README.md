## Introduction
tool-youtube-dl is a tool making [youtube-dl](https://github.com/ytdl-org/youtube-dl) easier to use. 

## Features
* Support reading from file 



## Installation
Put tool-youtube-dl , youtube-dl , ffmpeg in the same directory.
